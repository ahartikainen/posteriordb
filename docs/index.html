PosteriorDB Explorer

Browse models, posteriors, data metadata, reference draws, and bibliography entries from a build-time index. This page never fetches zipped datasets or draw archives in the browser. It shows metadata, previews, and links, while keeping the page pleasantly compact and not behaving like a caffeinated octopus.
